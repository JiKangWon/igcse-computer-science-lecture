1. Introduction to 2D Arrays

If a 1D array is like a single row of mailboxes, a 2D array (or two-dimensional array) is like a grid, a spreadsheet, or a game board. It has both rows and columns.

Think of a movie theater with rows and seats, a chessboard, or a matrix in mathematics. These are all 2D structures.

Indexing

To access a specific item, you now need two indices:

  • The first index is the row index.
  • The second index is the column index.

Just like 1D arrays, both indices almost always start from zero. We write the notation as arrayName[row][column].

For example, given a 3x3 array named grid:

   (col 0) (col 1) (col 2)
(row 0) [ "A" ] [ "B" ] [ "C" ]
(row 1) [ "D" ] [ "E" ] [ "F" ]
(row 2) [ "G" ] [ "H" ] [ "I" ]

grid[0][0] is "A" (row 0, col 0)
grid[1][2] is "F" (row 1, col 2)
grid[2][1] is "H" (row 2, col 1)

2. 2D Array Operations in Pseudo Code

The keywords are similar, but we must account for both dimensions.

2.1 Declaration & Initialization

You must specify the number of rows and columns. Here's how to create an array with 3 rows and 4 columns.

// 1. Declare an empty 2D array (3 rows, 4 columns)
DECLARE scoreSheet AS Array[3][4] OF Integer

// 2. Declare and initialize with values (a 2x3 array)
//    Note the nested "inner" arrays for each row
INITIALIZE ticTacToe = [ ["X", "O", "X"], 
                       ["O", "X", "O"], 
                       ["X", "O", "O"] ]

2.2 Accessing & Modifying Elements

You use the [row][column] index to read or change a value at a specific position.

INITIALIZE gameBoard = [ ["-", "-", "-"], 
                       ["-", "-", "-"], 
                       ["-", "-", "-"] ]

// Set an 'X' in the center (row 1, col 1)
SET gameBoard[1][1] = "X"

// Read the value from the top-right corner (row 0, col 2)
SET playerMove = gameBoard[0][2]

PRINT playerMove  // This will display "-"

3. Common Examples

The most common operation with a 2D array is "traversing" (looping through) it using nested loops (one loop inside another).

  • The outer loop iterates through the rows.
  • The inner loop iterates through all columns *for the current row*.

Example 1: Traversing and Printing All Elements

This pseudo code prints every element in a 2x3 matrix, one row at a time.

START
    INITIALIZE matrix = [ [1, 2, 3], 
                     [4, 5, 6] ]
    
    // This array has 2 rows (index 0 and 1)
    // This array has 3 columns (index 0, 1, and 2)
    
    // Outer loop: iterates through rows
    FOR row = 0 TO 1
        
        // Inner loop: iterates through columns
        FOR col = 0 TO 2
            
            // Print the element at [row][col]
            PRINT matrix[row][col]
            
        ENDFOR
        
        // (Optional) Print a new line after each row is finished
        PRINT_NEW_LINE()
        
    ENDFOR
END

Output:
1
2
3
(New Line)
4
5
6
(New Line)

Example 2: Calculating the Sum of All Elements

This pseudo code finds the total sum of all numbers in the 2D array.

START
    INITIALIZE myGrid = [ [10, 20], 
                     [30, 40], 
                     [50, 60] ] // A 3x2 array
    
    SET totalSum = 0
    
    // Outer loop: iterate rows (0 to 2)
    FOR row = 0 TO 2
        
        // Inner loop: iterate columns (0 to 1)
        FOR col = 0 TO 1
            
            // Add the value at [row][col] to the sum
            SET totalSum = totalSum + myGrid[row][col]
            
        ENDFOR
    ENDFOR
    
    PRINT "The total sum is: " + totalSum
END

Output:
The total sum is: 210

4. Sample Exercises

Exercise 1: Find the Largest Number

You are given an array: data = [ [5, 12, 3], [1, 9, 8], [15, 7, 2] ]. Write pseudo code to find the largest number in this 2D array and print it.

START
    INITIALIZE data = [ [5, 12, 3], [1, 9, 8], [15, 7, 2] ]
    
    // Assume the first element is the largest (to start)
    SET maxNumber = data[0][0]
    
    // Loop through rows (0 to 2)
    FOR row = 0 TO 2
        
        // Loop through columns (0 to 2)
        FOR col = 0 TO 2
            
            // Compare current element to our current max
            IF data[row][col] > maxNumber THEN
                // If it's bigger, update maxNumber
                SET maxNumber = data[row][col]
            ENDIF
            
        ENDFOR
    ENDFOR
    
    PRINT "The largest number is: " + maxNumber
END

Exercise 2: Calculate Row Sums

You are given an array: grades = [ [80, 85, 90], [92, 88, 95] ]. Write pseudo code that calculates the sum for each row (i.e., for each student) and prints each sum separately.

START
    INITIALIZE grades = [ [80, 85, 90], [92, 88, 95] ]
    
    // Loop through rows (0 to 1)
    FOR row = 0 TO 1
        
        // Reset the sum for each new row (each student)
        SET rowSum = 0
        
        // Loop through columns (0 to 2)
        FOR col = 0 TO 2
            SET rowSum = rowSum + grades[row][col]
        ENDFOR
        
        // Print the sum for the current row
        PRINT "Sum for Row " + row + " is: " + rowSum
        
    ENDFOR
END

Output:
Sum for Row 0 is: 255
Sum for Row 1 is: 275
