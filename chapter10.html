II. INTRODUCTION TO BOOLEAN LOGIC

Electronic circuits in modern devices are constructed from thousands of logic gates. A logic gate is an electronic circuit that relies on 'on/off' logic, accepting binary inputs and producing a binary output.

A logic circuit is formed when several logic gates are combined to carry out a specific function. The process of checking the output is performed using a truth table.

Note on Boolean Algebra: Boolean algebra is linked to logic circuits. The symbol · (dot) represents the AND operation. The symbol + (plus) represents the OR operation. A bar above the letter (e.g., A) represents the NOT operation.

IV. LOGIC CIRCUITS AND EXPRESSIONS

When logic gates are combined, they form a logic circuit. We must be able to write the logic expression from problems or truth tables.

A. Generating Expressions from Truth Tables

We only focus on rows where the output (X) is 1.

  • Case 1 (Single output 1): If X=1 only when A=1 and B=0, the expression is A AND NOT B.
  • Case 2 (Multiple outputs 1): We write an expression for each combination and join them with OR.
    If X=1 when (A=0 AND B=1) OR when (A=1 AND B=1), the expression is:
    (NOT A AND B) OR (A AND B)

B. Generating Expressions from Problem Statements

Scenario: An alarm, X, sounds (X=1) if input A is ON and input B is OFF; or if input B is ON and input C is OFF.

  • Analysis Part 1: (A is ON and B is OFF) $\rightarrow$ A AND NOT B
  • Analysis Part 2: (B is ON and C is OFF) $\rightarrow$ B AND NOT C
  • Logic Expression: (A AND NOT B) OR (B AND NOT C)

Two important logic circuits used in computers for binary addition.

1. Half Adder

The simplest circuit for binary addition. It performs addition on 2 bits, generating two outputs: Sum (S) and Carry (C). (e.g., $1 + 1 = 10$, where S=0 and C=1).

VI. SUMMARY OF KEY TERMS

A method of checking the output from a logic circuit; it lists all the possible binary input combinations and their associated outputs.
Term Definition
Logic gate An electronic circuit that relies on ‘on/off’ logic; the most common gates are NOT, AND, OR, NAND, NOR and XOR.
Logic circuit Formed from a combination of logic gates and designed to carry out a particular task; the output will be 0 or 1.
Truth table
Boolean algebra A form of algebra linked to logic circuits and based on TRUE or FALSE.

Analogy

Imagine a logic circuit as a complex security system protecting a vault. Each logic gate is a specific rule or sensor. The AND gate is a rule requiring two different keys to be turned simultaneously. The OR gate is a rule where either the manager's key or the guard's key will open a certain lock. The truth table is a comprehensive checklist that maps every possible combination of key inputs (A, B, C) to determine if the vault (X) opens (1) or stays locked (0).
