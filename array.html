1. Introduction to Arrays

An array is a fundamental data structure. Think of it as a row of numbered mailboxes, where each mailbox can hold one item. All items stored in an array must be of the same data type (e.g., all numbers, or all strings).

To access a specific item, you use its "mailbox number," which we call an index. In computer science, we almost always start counting from zero.

  • The first item is at index 0.
  • The second item is at index 1.
  • ...and so on.

Pseudo code is just a human-readable "blueprint" for our program logic. When we work with arrays in pseudo code, we focus on the logic of *how* to use them, not the strict syntax of a specific programming language.

2. Array Operations in Pseudo Code

We use simple keywords to describe array operations. The exact words can vary (e.g., DECLARE vs. INITIALIZE), but the meaning is the same.

2.1 Declaration & Initialization

This is how we create an array. We can create an empty array of a specific size, or create one with values already in it.

// 1. Declare an empty array named 'scores' that can hold 5 Integers
DECLARE scores AS Array[5] OF Integer

// 2. Declare and initialize an array with values
INITIALIZE studentNames = ["Alice", "Bob", "Charlie", "David"]

// 3. In many pseudo code styles, this is also acceptable
SET grades = [95, 88, 76]

2.2 Accessing Elements (by Index)

We use the array's name and the element's index in square brackets [] to read its value.

INITIALIZE studentNames = ["Alice", "Bob", "Charlie"]

// Access the element at index 1
SET secondStudent = studentNames[1]  // secondStudent is now "Bob"

// Print the element at index 0
PRINT studentNames[0]  // This will display "Alice"

2.3 Modifying Elements (by Index)

We can also use the index to change the value at a specific position.

INITIALIZE scores = [80, 85, 90]

// Print the original value at index 2
PRINT scores[2]  // This will display "90"

// Change the value at index 2
SET scores[2] = 95

// Print the new value
PRINT scores[2]  // This will display "95"

3. Common Examples

The most common array operation is "traversing" (looping through) it to process every element.

Example 1: Traversing and Printing All Elements

This pseudo code prints every name in the studentNames array.

START
    INITIALIZE studentNames = ["Alice", "Bob", "Charlie", "David"]
    
    // We loop from the first index (0) to the last index (3)
    FOR i = 0 TO 3
        // Access and print the element at the current index 'i'
        PRINT studentNames[i]
    ENDFOR
END

Output:
Alice
Bob
Charlie
David

Example 2: Calculating the Sum of an Array

This pseudo code finds the total sum of all numbers in the myNumbers array.

START
    INITIALIZE myNumbers = [10, 20, 30, 40]
    
    // Create a variable to hold the running total
    SET totalSum = 0
    
    // Loop from index 0 to 3
    FOR i = 0 TO 3
        // Add the number at the current index to our total
        SET totalSum = totalSum + myNumbers[i]
    ENDFOR
    
    // Print the final result
    PRINT "The total sum is: " + totalSum
END

Output:
The total sum is: 100

4. Sample Exercises

Exercise 1: Find the Largest Number

You are given an array of numbers: scores = [85, 92, 78, 95, 88]. Write pseudo code to find the largest number in this array and print it.

START
    // Initialize the array
    INITIALIZE scores = [85, 92, 78, 95, 88]
    
    // Assume the first element is the largest (for now)
    SET maxScore = scores[0]
    
    // Loop through the *rest* of the array (starting from index 1)
    FOR i = 1 TO 4
        // Check if the current element is bigger than our current max
        IF scores[i] > maxScore THEN
            // If it is, update maxScore to this new value
            SET maxScore = scores[i]
        ENDIF
    ENDFOR
    
    // After the loop, maxScore holds the largest value
    PRINT "The largest score is: " + maxScore
END

Exercise 2: Count Occurrences

You are given an array: grades = ['A', 'B', 'C', 'A', 'B', 'A']. Write pseudo code that counts how many times the grade 'A' appears in the array and prints the count.

START
    INITIALIZE grades = ['A', 'B', 'C', 'A', 'B', 'A']
    
    // Initialize a counter variable to 0
    SET countA = 0
    
    // Loop through the entire array (index 0 to 5)
    FOR i = 0 TO 5
        // Check if the element at the current index is 'A'
        IF grades[i] == 'A' THEN
            // If it is, increment the counter
            SET countA = countA + 1
        ENDIF
    ENDFOR
    
    // Print the final count
    PRINT "The grade 'A' appears " + countA + " times."
END
